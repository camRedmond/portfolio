# Assets Folder

This folder contains 88x31 pixel images and GIFs for the website.

## 88x31 Badges

The classic 88x31 pixel badge format, popularized in the 90s web era.

### Usage

Place your 88x31 images/GIFs in this folder, then reference them in the HTML:

```html
<img src="assets/your-badge.gif" alt="Description" width="88" height="31">
```

### Where to Find 88x31 Badges

- **https://cyber.dabamos.de/88x31/** - Huge collection of retro badges
- **https://anlucas.neocities.org/88x31Buttons** - Modern and retro badges
- **Create your own** at https://hekate2.github.io/buttonmaker/

### Recommended Badges for Developer Portfolio

- Browser/tech badges (Firefox, Chrome, HTML5, CSS3)
- Social badges (GitHub, Twitter, etc.)
- "Built with" badges (React, Node.js, etc.)
- Personal statement badges ("Code with ❤️", etc.)
- 512KB Club badge

### Size Considerations

- GIF: ~2-5KB each
- JPEG: ~1-3KB each
- Keep total under 100KB for all badges combined
